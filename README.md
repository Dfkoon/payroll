# Jsalamein Travel & Tourism 🌍💎

**A Premium Travel Experience Platform**

Welcome to the future of travel management. **Jsalamein Travel** is a world-class tourism platform designed to deliver luxury, ease, and cinematic beauty to travelers while providing a powerful command center for business owners.

---

## ✨ Premium Features

### 🎨 Cinematic User Experience
- **Glassmorphic Design:** A state-of-the-art interface using silk-smooth gradients and backdrop blurs.
- **Unified 12-Column Grid:** Optimized for PC and Mobile to ensure balanced, luxurious content distribution.
- **Custom Animations:** Hand-crafted transitions and reveal-up effects for a premium feel.

### 📊 Admin Control Center (The Heart)
- **Real-Time Analytics:** Track revenue, bookings, and customer trends via a futuristic dark-themed dashboard.
- **Tour Universe:** Manage inventory, update prices, and toggle "Featured" status with ease.
- **Reservation Pulse:** Complete tracker to manage payment statuses and booking details.

### 🏨 Travel Customization
- **Hotel Tier Selection:** Dynamic pricing for 3-star, 4-star, and 5-star hotel upgrades.
- **Multilingual Support:** Full English and Arabic support throughout the platform.
- **Stripe & Tawk.to Ready:** Infrastructure ready for live payments and real-time customer support.

## 🛠 Tech Stack

- **Backend:** [Laravel 11](https://laravel.com)
- **Frontend:** [Vue.js 3](https://vuejs.org) + [Inertia.js](https://inertiajs.com)
- **Styling:** [Tailwind CSS](https://tailwindcss.com)
- **Database:** MySql / PostgreSQL
- **Icons:** FontAwesome 6 + Custom SVGs

## 🚀 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Dfkoon/payroll.git
   cd payroll
   ```

2. **Install PHP Dependencies:**
   ```bash
   composer install
   ```

3. **Install JS Dependencies:**
   ```bash
   npm install
   ```

4. **Environment Setup:**
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

5. **Database Migration:**
   ```bash
   php artisan migrate --seed
   ```

6. **Run Development Servers:**
   ```bash
   npm run dev
   # In a separate terminal
   php artisan serve
   ```
